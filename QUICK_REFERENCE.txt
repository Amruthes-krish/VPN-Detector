📋 7 FEATURES AT A GLANCE
═══════════════════════════════════════════════════════════════════════════════

1️⃣  NETWORK & OWNERSHIP DETAILS
    ├─ ISP Name, Organization
    ├─ ASN (Autonomous System Number)
    ├─ Country, Region, City
    ├─ Latitude, Longitude, Timezone
    └─ Data Reliability Score
    
2️⃣  REPUTATION & ABUSE HISTORY
    ├─ Overall Reputation (excellent/good/neutral/poor/dangerous)
    ├─ Fraud Score (0-100)
    ├─ Spam Score (0-100)
    ├─ Blocklist Detection
    ├─ Abuse Report History
    └─ Threat Intel Matches

3️⃣  ASSOCIATED INFRASTRUCTURE
    ├─ Reverse DNS Records
    ├─ Associated Domains
    ├─ Known Subdomains
    ├─ SSL/TLS Certificate Data
    │   ├─ Issuer & Subject
    │   ├─ Validity Dates
    │   ├─ Subject Alternative Names
    │   └─ Self-signed Detection
    └─ Hostname Patterns

4️⃣  PASSIVE EXPOSURE (Shodan/Censys)
    ├─ Last Scanned Date
    ├─ Open Ports & Services
    ├─ Service Fingerprints
    ├─ Vulnerability Count
    ├─ Exposure Level (low/medium/high)
    └─ Honeypot Probability

5️⃣  ANONYMIZATION & HOSTING INDICATORS
    ├─ VPN Detection (provider, confidence)
    ├─ Proxy Detection (type, confidence)
    ├─ TOR Exit Node Detection
    ├─ Hosting Provider Detection (AWS, Azure, etc.)
    ├─ Anonymization Score (0-100)
    └─ Supporting Evidence

6️⃣  RISK CLASSIFICATION
    ├─ Risk Level (critical/high/medium/low)
    ├─ Risk Score (0-100)
    ├─ Primary Risk Factors
    ├─ Corroborated Signals
    ├─ Detailed Justification
    ├─ Business Impact Assessment
    └─ Confidence Score (0-1)

7️⃣  INVESTIGATION LEADS & PIVOTS
    ├─ Related IPs (same ASN/subnet)
    ├─ Shared ASN Pivot
    ├─ Shared Domains
    ├─ Shared SSL Certificates
    ├─ APT Group Associations
    └─ Investigation Priority (critical/high/medium/low)

🔌 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

1. POST /api/advanced/intelligence
   └─ Complete 7-feature analysis → AdvancedIntelligence object

2. POST /api/advanced/reputation-report
   └─ Reputation & abuse history → Reputation + Risk

3. POST /api/advanced/infrastructure-map
   └─ Domains, reverse DNS, certs → Infrastructure data

4. POST /api/advanced/exposure-analysis
   └─ Open ports, services → Passive exposure data

5. POST /api/advanced/anonymization-check
   └─ VPN/Proxy/TOR detection → Anonymization indicators

6. POST /api/advanced/risk-assessment
   └─ Risk classification → Risk level + impact

7. POST /api/advanced/investigation-leads
   └─ Related IPs, domains, ASN → Pivots by priority

🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════════

Step 1: Install
  $ pip install -r requirements.txt

Step 2: Get Databases
  Download MaxMind GeoLite2 databases and place in ./mmdb/

Step 3: Run
  $ uvicorn main_enhanced:app --reload

Step 4: Test
  $ curl -X POST http://localhost:8000/api/advanced/intelligence \
      -H "Content-Type: application/json" \
      -d '{"ip":"8.8.8.8"}'

Step 5: Access UI
  Open: http://localhost:8000
